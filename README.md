## Installation

## Commands

開啟 Server

```
$ hexo server
```

部署

```
$ hexo deploy
```

清除快取

```
$ hexo clean
```

建立新文章

```
$ hexo new [layout] <title>
```

把 markdown 生成 html

```
$ hexo generate
```

參考來源: [Hexo 官網](https://hexo.io/zh-tw/docs/commands.html)
